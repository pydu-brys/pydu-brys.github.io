#####################################Pokemon.pdf#####################################

This is a course project for MIS 586 (Big Data Analytics) at the University of Arizona.
This project aimd to explore the Pokemon spawning pattern with temporal and spatial 
features, and co-occurrence behaviors with network metrics.

Those features and metrics were later utilized to develop a C5.0 Decision Tree prediction 
model for recommending the next most-likely spawned Pokemon for a given set of features.

######################################Stock.pdf######################################

This is a course project for MIS 507 (Web Mining) at the University of Arizona. This 
project aimed to forecast stock price trend in a long term fashion through stock 
features extracted from Quandl and sentiments extracted from Reddit world news. 

The model was trained on historical stock price data of 60 companies from 6 different 
industries combined with the top 25 news headlines on Reddit world news (on a daily 
basis for the last 8 years). These features were used to predict stock price trends 
through C5.0 Decision Tree with accuracy slightly greater than 70%.

#####################################Dialysis.pdf####################################

This is a course project for Statistical Consulting at the National Cheng Kung University. 
This project explores the 5-year medical patterns of individuals who undergoes dialysis
as treatment for renal failure after such patterns. 

Taiwan's National Health Insurance Research Data - LHID2000 is used to complete this study. 
Data visualization was leveraged to display spatial and temporal patterns, and a Lasso 
Logistic Regression was was also used to construct a prediction model.
